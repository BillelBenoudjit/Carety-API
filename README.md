# CSE New Website

## Requirements 
| Requirement                                 |
| ------------------------------------------- |
| [NODE](https://nodejs.org/en/download/) |
| [NPM] Included with Node.js Installation |


## Deploy locally 

1 - Clone:

2 - Install the necessary dependencies:
```bash
npm install --save
```

3 - Create your `.env` file from `.env.example` and add the environment variables:
```bash
cp .env.example .env 
```

4 - Run the server :
```bash
nodemon index
```
